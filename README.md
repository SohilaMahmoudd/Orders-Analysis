# Orders Analysis
An interactive Power BI dashboard using the **Adventure Works dataset**.

🔧 **Project Overview**
The main goal was to design a clean, dynamic, and user-friendly dashboard that provides meaningful insights into sales performance, orders, and customer behavior.

---

📊 **Key Features**

* **Data Modeling (Star Schema)**
  Built a structured data model using fact and dimension tables to improve performance and simplify analysis.

* **Measures Table**
  Created a dedicated table to organize all DAX measures, making the model more maintainable and scalable.

* **Interactive Filter Pane**
  Implemented a toggle button to show/hide a filter pane containing slicers for:

  * Date
  * Category
  * Product

* **Navigation Buttons**
  Added buttons to enhance user experience:

  * Switch between **Category Table** and **Subcategory Table**
  * Back button for easy navigation

* **Custom Tooltip**
  Designed a tooltip on the **Orders by Date** chart to display detailed insights such as **online orders for a selected date**.

* **Dashboard Design**
  Focused on clean layout, clear KPIs, and intuitive interactions to improve usability.
